## 📖 Wie funktioniert das? / How does it work?

```
DNS-Anfrage / DNS Request
        │
        ▼
  ┌──────────┐    ✅ Match
  │ Allowlist │──────────── Zugelassen / Allowed
  └──────────┘
        │ Kein Match / No match
        ▼
  ┌──────────┐    🚫 Match
  │ Blocklist │──────────── Gesperrt / Blocked
  └──────────┘
        │ Kein Match / No match
        ▼
   Upstream DNS ──────────── Zugelassen / Allowed
```

**DE:** Domains auf der Allowlist werden **immer** durchgelassen – auch wenn sie auf einer Blockliste stehen. Regelsyntax: `@@||domain^`

**EN:** Domains on the allowlist are **always** permitted – even if they appear on a blocklist. Rule syntax: `@@||domain^`
